# 🧠 YouTube AI Tutor

> An intelligent, RAG-powered learning assistant that transforms any YouTube video into an interactive study session — ask questions, get grounded answers, and learn faster.

---

## ✨ Features

- 🎬 **YouTube Transcript Fetching** — Automatically extracts transcripts in English, Hindi, or Bengali
- ✂️ **Smart Text Chunking** — Splits transcripts into overlapping chunks for accurate retrieval
- 🧠 **Multilingual Embeddings** — Uses `intfloat/multilingual-e5-large` for semantic understanding
- 🗂️ **FAISS Vector Store** — Lightning-fast similarity search over transcript chunks
- 🔍 **Contextual Compression** — MMR retrieval + LLM-based chunk filtering for precise context
- 💬 **AI-Powered Q&A** — Answers grounded in video content, generated by `google/gemma-3-27b-it`
- 📜 **Chat History** — Full conversation memory across the session
- 📚 **Transcript Context Panel** — Expandable view of the exact chunks behind each answer

---

## 🛠️ Tech Stack

| Layer | Tools |
|---|---|
| **Language** | Python 3.11+ |
| **Frontend** | Streamlit |
| **LLM** | LangChain + HuggingFace (`google/gemma-3-27b-it`) |
| **Embeddings** | HuggingFace Endpoint (`intfloat/multilingual-e5-large`) |
| **Vector Store** | FAISS |
| **Retriever** | MMR + Contextual Compression (LLMChainExtractor) |
| **Transcript Source** | `youtube-transcript-api` + `yt-dlp` (fallback) |

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/souryadip-official/AskTube-AI.git
```

### 2. Install dependencies

```bash
pip install streamlit youtube-transcript-api yt-dlp requests faiss-cpu \
  langchain-core langchain-classic langchain-community \
  langchain-huggingface huggingface-hub
```

### 3. Obtain a HuggingFace API key

| Key | Link |
|---|---|
| HuggingFace API Key | https://huggingface.co/settings/tokens |

> Make sure your token has **read** access and that you've accepted the model's license on HuggingFace if required.

### 4. Run the app

```bash
streamlit run app.py
```

---

## 🚀 Usage

1. Open the app in your browser (default: `http://localhost:8501`)
2. Enter your **HuggingFace API Key** in the sidebar and click **Connect**
3. Paste a **YouTube video link** in the main panel
4. Click **Process Video** — the app will fetch the transcript, chunk it, and build the vector store
5. Ask questions in the **Chat** panel on the sidebar
6. Expand **Transcript Context** under any answer to see the source chunks

---

## 💡 Notes

- Answers are grounded in the video's transcript. If the answer isn't in the video, the model will say so rather than hallucinate.
- Transcripts must be available and not disabled by the video owner. If `youtube-transcript-api` fails, the app automatically falls back to `yt-dlp` to extract captions.
- Supported transcript languages: **English**, **Hindi**, **Bengali** (auto-selected in order of availability).
- Python 3.11+ is recommended for full compatibility with all dependencies.
